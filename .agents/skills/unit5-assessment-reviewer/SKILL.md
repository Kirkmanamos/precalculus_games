## Review Lenses

Check each question for:

- alignment to sections 5.1-5.3
- non-honors difficulty
- exact unit-circle solvability
- fit with the five Trashketball categories

Flag questions that drift into:

- 5.4+ content
- multiple-angle solving
- calculator dependence
- graphing/modeling
- inverse trig challenge material
- non-standard exact values that students cannot get from the unit circle

## Output Shape

Lead with findings, ordered by severity when there are real problems.

Then summarize:

- which categories are well represented
- which categories are missing or underweighted
- what the game should emphasize to prepare students for the assessment

If the user asks for edits, preserve the LaTeX structure and teacher voice while tightening the assessment toward the approved non-honors scope.
