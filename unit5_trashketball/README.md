# Unit 5 Trashketball

A browser-based Trashketball review game for **Unit 5 sections 5.1-5.3** in non-honors precalculus.

## How to Run

1. Open this folder.
2. Double-click `Trashketball.html`.
3. The game opens in your browser with no build step or server.

An internet connection is needed on first load for MathJax and fonts.

## Scope

This board is designed to prepare students for the non-honors quick check on trig equations and identities using only standard unit-circle values.

The five categories are:

- Simplifying Trig Expressions with Fundamental ID's
- Verifying Trig Identities
- Solving Trig Equations
- Solving Trig Equations with Quadratic Forms
- Solving Trig Equations with Quadratics / Square Roots / Fundamental ID's

Difficulty ramps by point value:

- `$100-$200`: direct practice
- `$300-$400`: multi-step algebra and identity work
- `$500`: the heaviest question in the category
